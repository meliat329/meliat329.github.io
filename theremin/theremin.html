Raspberry Pi Theremin

Date: Spring 2021

Role:Designer and Producer

Course:Learning Interaction Design, Tangible Hardware

Skills: Python, SonicPi, Raspberry Pi

Sketch Idea

I began with sketching my initial project idea: a theremin made from ultrasonic distance sensors and a raspberry pi. The setup was simple with a sensor on each side so that, like a theremin, both pitch and volume could be controlled.

Feedback

I collected feedback from my classmates, and they pushed me to take the idea further and tie it to learning somehow.

More Sketching

I added a light system to the sketch that allows the user to input a note they would like to learn to play and the lights would respond to how close they were to the note. Red would be very far, yellow would be close, and green would be on the note.

Building and Coding

I began by building the simple input and output system on the raspberry pi. I then used Python and sonicPi to create the program that allows you to play music, and the separate learning mode that allows you to learn how to play specific notes.

Final Project

For the final project I created a short video talking about how the theremin was made and showing a short demo. If I was to take this project further, I would add another learning mode that teaches someone how to play songs on the theremin.
